This repository contains two files, grititem and gritweap.

grititem.json contains both armor and weapons from Grit and Glory in an original and complete state.

gritweapon.json contains weapons from Grit and Glory edited for personal reasons and otherwised balanced in different ways. It also lacks any wound dealing to enemies for the purpose of simplicity from a DM's perspective. Value is temporarily removed until 5etools can support alternate currency conversion systems.

WARNING: gritweapon.json is a personal project, unless you prefer it, always use grititem.

Content is based on Grit and Glory v5.08:
https://thetrove.net/Books/Dungeons%20&%20Dragons/3rd%20Party/5th%20Edition/Homebrew/Grit%20and%20Glory%20v5.pdf
