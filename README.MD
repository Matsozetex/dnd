This repository contains one JSON object for use with BetterR20-5eTools and 5eTools.

This is strictly a personal project, the actual grit and glory item list can be found at: https://github.com/TheGiddyLimit/homebrew.

For the purpose of streamlining combat, properties that usually give wounds or affect creatures that are wounded have different effects. Also for the sake of balancing some properties have been completely changed.

If using the silver standard system, remember to convert the values of these weapons as they use the standard currency systen.

Content is based on Grit and Glory v5.08:
https://thetrove.net/Books/Dungeons%20&%20Dragons/3rd%20Party/5th%20Edition/Homebrew/Grit%20and%20Glory%20v5.pdf

Note on how slots work, it is based on properties or practicality:
- Each weapon starts with a base slot value of 2 slots
- It increases by 1 if the weapon is two-handed, such as the Greatsword
- It decreases by 1 if the weapon can be folded (like a net), or is small, like a dagger
- It increases by 2 if the weapon has the tripod property or has an exceeding long load time (in the case of the tiger gun)
- It decreases by 2 if it is extremely small, like darts
